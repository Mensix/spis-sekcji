# Spis Sekcji JBwA i tag-grupek

Repozytorium z kodem źródłowym. Napisane

## Technologie
* Vue
* Quasar
* Firebase

## Instalacja na systemie Windows

1. `git clone https://github.com/Mensix/SpisSekcji.git`
2. `cd <lokalizacja>`
3. `npm install`
4. `npm run dev`
